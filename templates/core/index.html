Welcome at retrospectively!

You probably want to know, what this is about.

To make it short:

It's an idea inspired by quantified self and raspberries (yes, I was eating raspberries then)

I wanted following things:

  • retrospective once a week
  • get an email with link to the form
  • be able to browse my past retrospectives
  • retrospectives to be in a form easy for quantification
  • charts
  • simple statistics
  • easy export to csv, json, whatever
  • programmer api
  • whole platform be secure and private
  • sharability

How it is implemented?

  • django
  • Twitter bootstrap
  • django rest framework
