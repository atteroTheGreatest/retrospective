Shubadubudu, welcome at retrospectively!

You probably want to know, what this is about.

To make it short:

  • it's an idea inspired by quantified self and raspberries (yes, I was eating raspberries then)
  • i want to make retrospective once a week
  • there will be a form with fields like: "What did I learn?", "How is my progress in different fields?", "Conclusions", "Thoughts", and so on
  • i want to get an email with link to the form
  • i want to be able to browse my past retrospections
  • i want it to be in form easy for quantification
  • i want charts
  • i want simple statistics
  • i want easy export to csv, json, whatever
  • i want programmer api
  • i want it to be secure and private
  • i want to share, if it's nice and sharable
  • implementation
    • django
    • Twitter bootstrap
    • angular js

Let's take a journey!
